# -AI-Training-Data-System

## Overview
This project simulates an AI data management system using SQL, including datasets, models, training results, and a chatbot knowledge base.

## Features
- Dataset management
- AI model tracking
- Training results analysis
- Simple AI chatbot simulation (retrieval-based)

## Technologies
- SQL
- Database Design

## AI Chatbot Feature
The chatbot uses a knowledge base stored in SQL tables to return predefined answers and simulate AI behavior.
